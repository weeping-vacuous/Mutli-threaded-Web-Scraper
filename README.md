# Multi-Threaded Web Scraper

A high-performance web scraper built with Python that uses multi-threading and object-oriented programming principles to efficiently scrape book data from the [Books to Scrape](http://books.toscrape.com/) website.

## Features

- **Multi-threaded Architecture**: Leverages multiple threads to scrape pages concurrently and significantly reduce execution time
- **Object-Oriented Design**: Clean, modular code structure using OOP principles for maintainability and extensibility
- **BeautifulSoup4 Integration**: Utilizes BS4 for efficient HTML parsing and data extraction
- **Efficient Data Collection**: Scrapes book information including titles, prices, availability, and ratings

## Requirements

- Python 3.7+
- BeautifulSoup4
- Requests
